1 Cucumber Chef 2.0

Test Driven Infrastructure Notes

Book is still for 1.0, as he hasn't gotten the 2.0 book out.

cucumber-chef depends on ec2 because a t1.micro can be free or very cheap. It creates a LXC container on the instance and deploys the Cookbooks on the LXC Instance.

1.1 Getting started with Cucumber Chef

1.1.1 Install Cucumber-chef

gem install cucumber-chef

1.1.2 Initialize cucumber-chef to configure everything

cucumber-chef init

It will ask you many questions fill in the answers.

1.1.3 Setup your cucumber-chef cluster.

cucumber-chef setup

Note this will take a few minutes to complete, once it's up the environment will be up.

1.1.4 Creating your first test

cucumber-chef create beer

features/beer/beer.feature is created with a basic definition features/beer/step_features/beer_steps.rb has all the steps
